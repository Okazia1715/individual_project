# Data Preparation

This folder contains Python scripts used to clean and prepare datasets
for later analysis and modeling stages.

## Scripts

### 1) Image inversion + contrast adjustment

**Purpose:**  
Prepare histology images for downstream segmentation by applying:

- intensity inversion
- contrast scaling to 1.2

This preprocessing improves Cellpose performance for the next project
stage: training a model to segment adipocytes.

**Input dataset:**

- `1_datasets/Lung_and_Colon_Cancer/Colon_Benign_Tissue/invert/raw/`
  (raw images, unchanged)

**Processing steps:**

1. Load each image from the input folder
2. Invert image intensities (bright ↔ dark)
3. Apply contrast factor 1.2
4. Save processed images as a new dataset

**Output dataset created:**

- `1_datasets/Lung_and_Colon_Cancer/Colon_Benign_Tissue/invert/processed/`

**Script file:**

- `2_data_preparation/scripts/invert_contrast_1p2.py`

**Notes:**

- Raw datasets in `1_datasets/` must remain untouched.
- Outputs are deterministic and can be regenerated by re-running the script.
